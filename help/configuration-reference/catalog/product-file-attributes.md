## [!UICONTROL Configure Allowed File Types and Size]

![Atributos de archivo de producto](./assets/file-validation.png)

| Campo | [Ámbito](../../getting-started/websites-stores-views.md#scope-settings) | Descripción |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | Global | Lista separada por comas de los tipos de archivo permitidos, como `pdf,doc,docx,txt`. Déjelo en blanco para utilizar los tipos de archivo predeterminados permitidos: pdf, doc, docx, xls, xlsx, ppt, pptx, txt, csv, zip. |
| [!UICONTROL Maximum File Size] | Global | Tamaño máximo de archivo permitido en MB, como `20.0`. El límite predeterminado es 16 MB. El tamaño máximo de archivo permitido es 31 MB. |
